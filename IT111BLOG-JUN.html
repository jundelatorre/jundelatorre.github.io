What I have learned

Lesson 1

What is Computing?

Computing is the utilization of computer technology to process information, involving hardware, software, data manipulation, algorithms, programming, networking, and cybersecurity, with applications ranging from simple calculations to complex data analysis and problem-solving.

Lesson 2

Professions and Careers in Computing

Professions and Careers in Computing encompass roles such as software development, hardware engineering, information systems, network administration, cybersecurity, data science, web development, and artificial intelligence, providing diverse opportunities within the rapidly evolving field of computing and technology.

Lesson 3

Information Processing Cycle

The Information Processing Cycle is a series of stages—input, processing, output, and storage—through which a computer handles data for tasks like calculation, display, and storage.

Lesson 4

Computer Systems

A computer system is a combination of hardware and software components, including a central processing unit (CPU), memory, storage, input and output devices, and an operating system, working together to process data and perform tasks.

Lesson 5

Concept of Computer Security

The concept of computer security involves protecting systems and data from unauthorized access or harm through measures like encryption, access controls, and awareness of potential threats.

Lesson 6

HTML & CSS

HTML (Hypertext Markup Language) is used for structuring web content with tags, while CSS (Cascading Style Sheets) styles HTML elements to control layout and presentation, collectively forming the foundation for building visually appealing and structured web pages.

My Favorite Lesson

My favorite lesson is about the Concept of Computer Security, its because it taught me that the awareness of computer security is essential for individuals and to contribute a safer and more secure online environment.
